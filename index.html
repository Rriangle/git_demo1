使用說明

這是 6 頁的後台導覽 Demo。左側選單切換區塊，不用換頁面。每張卡與表格都放了假資料，之後把它們換成 API 即可。

  • 資料管理：遊戲 + 來源/指標（同頁）
  • 每日數據：查/編 game_metric_daily
  • 熱度＆快照：重算 → 產生 → 歷史
  • 洞察文章：posts + 快照
  • 社群管理：threads / posts / reactions / bookmarks
  • 系統工具：敏感詞 / 匯入匯出
